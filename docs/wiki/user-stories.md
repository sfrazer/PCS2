# User Stories

All items are MVP scope. Listed in priority order.

| # | As a player, I want to… | So that… |
|---|---|---|
| 1 | Open the application and see an 800×420 construction canvas with a palette | I can start designing immediately |
| 2 | Select a table element from the palette (flipper L/R, launcher, pop bumper, drop target, spinner) | I can choose what to place |
| 3 | Click the canvas to place the selected element | I can build the table layout |
| 4 | Click and drag a placed element to reposition it | I can adjust the layout |
| 5 | Rotate a placed element | I can orient elements correctly |
| 6 | Delete a placed element | I can correct mistakes |
| 7 | Save the table to a JSON file | I can preserve my work |
| 8 | Load a previously saved table from a JSON file | I can resume editing |
| 9 | Switch from construction mode to play mode | I can test the table |
| 10 | Control the left flipper with the Left Shift key | I can play pinball |
| 11 | Control the right flipper with the Right Shift key | I can play pinball |
| 12 | Launch the ball using the launcher | I can start a game |
| 13 | Return to construction mode from play mode | I can iterate on the design |
| 14 | Export the table as a portable JSON artifact | The table can be used in another game |
